21. contoh syntax callable dalam php


Sintaks Dasar PHP

Sebuah script PHP selalu Dimulai dengan blok. <?Php dan Diakhiri dengan ?> . A PHP scripting blok dapat ditempatkan di manapun dalam dokumen.

Pada server dengan dukungan steno diaktifkan, Anda dapat memulai blok scripting dengan <? dan diakhiri dengan >? .

Untuk kompatibilitas maksimum, kami sarankan Anda menggunakan formulir standar (<? Php) bukan pada bentuk singkatan .

<?php
?>

Sebuah file PHP biasanya berisi tag HTML, seperti sebuah file HTML, dan beberapa kode PHP scripting.

Di bawah ini, kita memiliki contoh sederhana script PHP yang mengirim teks “Hello World”ke browser :

<html>
<body><?php
echo “Hello World”;
?></body>
</html>

Setiap baris kode PHP harus Diakhiri dengan titik koma. Titik koma adalah pemisah dan digunakan untuk membedakan satu set instruksi dari yang lain.


22. bagaimana penulisan pembatas bahasa pemrograman PHP ? berikan contohnya !​


Penulisan pembatas bahasa pemrograman PHP adalah dengan cara membukanya dengan <? dan menutupnya dengan ?>. Contohnya adalah

<?php

   echo "Halo dunia";

?>

Penjelasan:

PHP hanya mengeksekusi kode yang ditulis dengan pembatas yang didefinisikan dalam  sintaks PHP dasar. Apa pun di luar pembatas tidak akan diproses oleh PHP. PHP (PHP: Hypertext Preprocessor) adalah  bahasa pemrograman skrip sisi server open source yang mengeksekusi instruksi pemrograman saat dijalankan sebagai bahasa skrip. Tentu saja, hasil instruksi  tergantung pada data yang diproses. Karena PHP adalah bahasa pemrograman sisi server, skrip diproses oleh PHP  di server. Jenis server yang sangat biasa digunakan bersama dengan PHP antara lain Apache, Nginx, dan LiteSpeed.

Bahasa Pemograman Yang Dipilih: PYTHON

1. Fungsi print()

Fungsi print() digunakan untuk menampilkan teks. Dalam penggunaannya fungsi print harus menggunakan tanda kurung (Di Python 3) sedangkan di Python 2 Fungsi print Tidak memakai tanda kurung

Contoh Penggunaan:

print('Saya Belajar') #menampilkan teks Saya Belajar

print("Saya Makan") #menampilkan teks Saya Makan

Di fungsi print ini bisa menggunakan single quotes (') ataupun double quotes (")

2. Fungsi str()

Fungsi str() digunakan untuk mengkonversikan tipe data lain ke tipe data string, contoh penggunaannya yaitu:

#Contoh 1

a = 26 #Buat variabel bertipe data int (bil bulat)

print(type(a)) #menampilkan tipe data variabel a (outputnya berupa = <class = 'int'> yang berarti variabel a bertipe data int (bil bulat)

#Contoh 2

b = True #Buat variabel bertipe data boolean

print(type(str(b))) #menampilkan tipe data variabel b yang sudah dikonversikan menjadi string (outputnya berupa <class = 'str'> yang berarti variabel b bertipe data string)

3. Fungsi type()

Fungsi type() digunakan untuk mengetahui tipe data suatu variabel (bisa langsung tanpa variabel), contoh penggunaannya yaitu:

#Contoh 1

a = 'Beli sepeda kapan kapan' #buat variabel a yang value nya Beli sepeda kapan kapan

print(type(a)) #menampilkan tipe data a (outputnya <class = 'str'> yang berarti variabel a bertipedata string)

#Contoh 2

print(type(False)) #menampilkan tipe data False (outputnya berupa <class = 'bool'> yang berarti False bertipe data Boolean)

4. Fungsi abs()

Fungsi abs() digunakan untuk mengubah suatu bilangan negatif menjadi bilangan positif, jika yang diinput di parameternya selain angka maka akan mengeluarkan error 'TypeError' , contoh penggunaannya:

num = -68 #buat variabel bernama num yang valuenya bilangan negatif

print(abs(num)) #menampilkan variabel num yang sudah menjadi bil positif (outputnya: 68)

5. Fungsi list()

fungsi list() digunakan untuk mengkonversikan string ke list, jika parameter yg dimasukan bukan bertipe data string maka akan mengeluarkan error 'TypeError', contoh penggunaannya yaitu:

kata = "Bruh" #buat variabel bertipe data string.

print(list(kata)) #menampilkan variabel kata yang dikonversikan menjadi list (perhuruf) (outputnya yaitu: ['B', 'r', 'u', 'h'])

6. Fungsi float()

Fungsi float() digunakan untuk mengkonversikan angka ke bilangan desimal, jika yang diinput ke fungsi ini selain angka maka akan mengeluarkan error 'TypeError', contoh penggunaannya yaitu:

a = 7 #buat variabel a tipe data int

print(float(a)) #menampilkan variabel a yg sdh di ubah ke desimal (outputnya 7.0)

7. Fungsi isalpha()

Fungsi isalpha() digunakan untuk mengecek apakah parameter yg dinputnya isinya huruf semua. contoh penggunaannya:

#Contoh 1

print("Laboratorium".isalpha()) #menampilkan apakah benar "Laboratorium" itu huruf semua (outputnya True)

#Contoh 2

print("4".isalpha()) #menampilkan apakah benar "4" itu huruf semua (outputnya False Karena "4" walaupun tipe data nya string tapi isinya 4 (angka))

8. Fungsi isalnum()

Fungsi isalnum() digunakan untuk mengecek apakah parameter yg dinputnya isinya huruf atau angka. contoh penggunaannya:

#Contoh 1

print("Laboratorium".isalnum()) #menampilkan apakah benar "Laboratorium" terdiria itu huruf atau angka (outputnya True)

#Contoh 2

print("4".isalnum()) #menampilkan apakah benar "4" itu huruf atau angka (outputnya True Karena fungsi isalnum mengecek Huruf ATAU Angka (jika salah satu benar maka nilainya True))

#Contoh 3

print("Boba24".isalnum()) #menampilkan apakah benar "Boba24" terdiri dari angka atau huruf (Outputnya True karena ada angka, dan ada huruf)

9. Fungsi exec()

Fungsi exec() berfungsi untuk mengeksekusi input yg diberikan, contoh:

a = "print('Balon')" #buat variabel a

exec(a) #mengeksekusi command yg ada di variabel a yaitu print('Balon') (outputnya Balon)

10. Fungsi int()

Fungsi int() digunakan untuk mengkonversikan bilangan ke bilangan bulat (integer), contoh:

a = 8.9 #buat variabel a bertipe data float

print(int(a)) #menampilkan variabel a yang sdh dikonversikan tipe datanya ke integer (bil bulat) (Outputnya 8)

NOTE:

Python itu Case-Sensitive yang artinya 'A' beda dengan 'a'Jawaban Saya tidak COPAS

  function is_expired($expired_date) {
  list($year, $month, $day) = explode('-', $expired_date);
  $new_expired_date = sprintf('%04d%02d%02d', $year, $month, $day);
  $date_now = date("Y-m-d");
    if ($new_expired_date > $date_now) {
    return false;
    } else {
    return true;
