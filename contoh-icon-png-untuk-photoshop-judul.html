17. Apa fungsi dari icon kontrol menu dan pallete pada Photoshop?​


Jawaban:

1. Palet Options/Options Bar fungsi untuk menunjukan perintah-perintah yang tersedia pada options bar untuk tool yang sedang sobat gunakan.

2. Palet Layer berfungsi untuk menunjukan semua layer yang ada di dalam lembar kerja kita (tempat objek).

3. Palet Channel berfungsi sebagai tempat informasi tentang warna yang berlaku pada gambar yang dikerjakan.

4. Palet History berfungsi untuk menyimpan status atau history pekerjaan.

5. Palet Action berfungsi untuk merekan ataupun mengulang kembali perkerjaan yang sobat kerjakan.

6. Palet Navigator berfungsi sebagai preview thumbnail file image yang sedang dibuka.

7. Palet Info berfungsi untuk menampilkan semua informasi yang berhubungan dengan tampilan lembar kerja.

8. Palet Swatches merupakan sebuah tabel warna.

9. Palet Color berfungsi untuk mengatur warna Forground atapun Background.

10. Palet Styles berisi kotak yang menyediakan beberapa pilihan style yang dapat gunakan pada teks maupun gambar.

29. 4 jenis icon T yang digunakan untuk membuat tulisan atau teks pada toolbox adobe photoshop​


Jawaban:

1. Move Tool

Move Tool

Move Tool (shortcut keyboard: L) berfungsi untuk memindahkan layer/objek baik itu berupa gambar ataupun teks.

2. Marquee Tool

Marquee Tool

Marquee Tool berfungsi untuk menyeleksi objek. Marquee Tool dibagi menjadi 4, yaitu:

Rectangular Marquee Tool (shortcut keyboard: M), berfungsi untuk menyeleksi objek berbentuk kotak/segiempat.

Elliptical Marquee Tool (shortcut keyboard: M), berfungsi untuk menyeleksi objek berbentuk lingkaran/elips.

Single Row Marquee Tool, berfungsi untuk menyeleksi objek dengan garis horizontal.

Single Column Marquee Tool, berfungsi untuk menyeleksi objek dengan garis vertikal.

3. Lasso Tool

Lasso Tool juga berfungsi untuk menyeleksi objek, tetapi dalam bentuk yang bebas. Lasso Tool dibagi menjadi 3, yaitu:

Lasso Tool (shortcut keyboard: L), berfungsi untuk menyeleksi objek dalam bentuk bebas sesuai keinginan.

Polygonal Lasso Tool (shortcut keyboard: L), berfungsi untuk menyeleksi objek dalam bentuk garis-garis bebas.

Magnetic Lasso Tool (shortcut keyboard: L), menyeleksi objek dengan garis yang otomatis menempel pada objek yang diseleksi.

4. Quick Selection & Magic Wand Tool

Quick Selection Tool (shortcut keyboard: W), berfungsi untuk menyeleksi area secara bebas berdasarkan warna atau pixel.

Magic Wand Tool (shortcut keyboard: W), berfungsi untuk menyeleksi objek berdasarkan warna yang sama.

4. 3cara Sobat Yaitu:

1.Save

Kalau yang satu ini sih, saya yakin semuanya sudah pada tau ya.Yap,cara yang pertama ini kita menggunakan menu Save yang berada di menu File->Save atau bisa juga menggunakan shortcut Ctrl+S. Sayangnya, jika sobat menggunakan menu ini sobat hanya bisa melakukan proses penyimpanan dengan format yang sama (contoh : jika sobat membuka PSD, maka akan disimpan menjadi PSD juga).

2.Save As

Lanjut ke cara yang kedua yaitu melalui menu Save As yang berada di File->Save As atau bisa menggunakan shortcut Shift+Ctrl+S. Perintah save as ini digunakan untuk menyimpan file dengan format, nama, dan lainnya berbeda dengan format aslinya. (contoh : jika sobat membuka PSD, maka sobat bisa mengubah formatnya menjadi JPEG dengan cara ini).

3.Export
