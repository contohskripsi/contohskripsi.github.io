7. Bagaimana cara meng-import gambar dari autocad ke sketchup?


Copy gambar yang ingin anda import ke file baru agar gambar yang tidak perlu tidak memberatkan sketchup. seperti pada gambar saya hanya mengambil gambbar denah.Hapus layer selain layer tembok dan kolom.Explode semua gambar hal ini sangat penting agar memudahkan langkah selanjutnya di sketchup. Selain itu jika anda biasanya membuat tembok dengan multi line, jika tidak anda explode maka tidak akan terimport di sketchup.Bukua sketchup anda kemudian klik pada menu File>ImportSetelah muncul window seperti diatas pilih jenis filenya yaituAutoCAD files (*.dwg, *.dfx).Sebelum menekan open klik pada Options terlebih dahulu. Pada bagian Scale pilih unit yang anda gunakan dalam file DWG tersebut. Saat pertama kali unit defaultnya adalah feet, jika tidak anda atur maka ukuran pada sketchup tidak akan sesuai dengan ukuran di AutoCAD.Kemudian tekan Open maka gambar akan terimoprt ke sketchup.Gambar yang anda import akan berada dalam satu grup. Langkah berikutnya adalah membuat face atau bidang pada denah tersebut. Pertama download dulu plugin make_faces.rb kemudian instal. cara menginstalnya bisa anda lihat disini. setelah itu masuk ke group dan pilih semua lalu klik kanan dan pilih make face, jika tidak ada pilihan make face anda bisa membuat shortcutnya caranya bisa anda lihat disini.setelah selesai hasilnya akan seperti berikut.

9. Apa yang anda ketahui tentang penggambaran tiga dimensi dengan menggunakan aplikasi Autocad?tolong dijawab kak, makasih​


Jawaban:

AutoCAD adalah perangkat lunak komputer CAD untuk menggambar 2 dimensi dan 3 dimensi yang dikembangkan oleh Autodesk. Keluarga produk AutoCAD, secara keseluruhan, adalah software CAD yang paling banyak digunakan di dunia.

Penjelasan:

AutoCAD digunakan oleh insinyur sipil, land developers, arsitek, insinyur mesin, desainer interior dan lain-lain.

Format data asli AutoCAD, DWG, dan yang lebih tidak populer, Format data yang bisa dipertukarkan (interchange file format) DXF, secara de facto menjadi standard data CAD. Akhir-akhir ini AutoCAD sudah mendukung DWF, sebuah format yang diterbitkan dan dipromosikan oleh Autodesk untuk mempublikasikan data CAD.

AutoCAD saat ini hanya berjalan disistem operasi Microsoft. Versi untuk Unix dan Macintosh sempat dikeluarkan tahun 1980-an dan 1990-an, tetapi kemudian tidak dilanjutkan. AutoCAD masih bisa berjalan di emulator seperti Virtual PC atau Wine.

AutoCAD dan AutoCAD LT tersedia dalam bahasa Inggris, Jerman, Prancis, Italia, Spanyol, Jepang, Korea, Tionghoa Sederhana, Tionghoa Tradisional, Rusia, Ceko, Polandia, Hongaria, Brasil, Portugis, Denmark, Belanda, Swedia, Finlandia, Norwegia dan Vietnam.

15. Cara menggambar rumah di Autocad


Jawaban:

Membuat kapling/denah biasa terlebih dahulu.

Banyak cara yang di gunakan dalam membuat denah rumah ini bisa dengan membuat layernya terlebih dahulu atau dengan command multiline, tapi saya akan menggunakan command yang sudah saya berikan.

command yang di pakai : Line (L), Offset (O)

Gambarlah desain awalnya terlebih dahulu sesuai dengan ukuran yang sudah ada di gambar ini

Untuk ukuran dinding 15 mm

Membuat pintu dan jendelanya di tempat/space yang terpisah dari denah yang di buat tadi untuk lebih gampang di pindahkan nantinya.

command yang di pakai : Rectangle (Rec), Arc, Copy (Co)

Gambarlah sesuai dengan ukuran yang sudah di sediakan seperti gambar di bawah ini

Untuk gambar jendela yang di kotak merah

Memindahkan object pintu yang sudah di buat tadi ke object denah/kapling yang akan di buatkan pintu.

command yang di pakai : Copy (Co), Mirror (Mi), Rotasi (Ro)

Ubah tampilan object pintunya seperti yang di kotak merah dengan menggunakan command Mirror dan Rotasi

Lalu pindahkan object pintu ke space yang dimana akan di jadikan tempat keluar masuk seperti yang di lingkaran merah dengan menggunakan command Copy

sekarang giliran Memindahkan object jendelanya ke space yang menurut kalian cocok untuk di pasang jendela.

command yang di pakai : Copy (Co)

pindahkan object jendela yang sudah di buat sebelumnya (dalam kotak merah) ke space yang di inginkan (dalam lingkaran merah)

anda bisa membuat garis bantu agar mendapatkan midpoint/basepointnya untuk menyesuaikan letaknya

Membuat toilet.

Agar lebih mudah kita membuat toiletnya berada di luar denah, jadi nanti tinggal di pindahkan saja

command yang di pakai : Rec (R), (Offset (O), Arc, Mirror (Mi)

Gambarlah toilet sesuai ukuran seperti gambar di bawah ini dengan menggunakan command2 tersebut atau sesuai keinginan anda juga ga apa2

memindahkan toilet yang sudah di buat tadi ke kamar mandi.

command yang di pakai : Move (M)

pindahkan toilet tadi seperti gambar di bawah ini dengan menggunakan command Move

Membuat bak mandi.

command yang di pakai : Rectangle (Rec)

buatlah bak mandi dengan ukuran yang anda inginkan

membuat teras depan dan belakang dan lantai kamar mandi.

command yang di pakai : Line (L), Hatch (Ha)

membuat areanya terlebih dahulu dengan menggunakan command Line

Lalu memberi tampilan seperti lantai dengan menggunakan command Hatch dengan pengaturan pattern = NET, scale = 300, dan add pick points sesuai gambar di bawah ini

Untuk lantai kamar mandi ukuran scale = 150

Membuat carport/garasi.

command yang di pakai : Line (L), Offset (O), Copy (Co)

Buatlah garasi/carport dengan command2 tersebut seperti gambar di bawah ini atau sesuai keinginan anda

Memberi nama pada masing-masing bagian.

command yang di pakai : Dtext (Dt)

Buatlah nama pada masing-masing bagian dengan menggunakan command Dtext tentunya

Jawaban:

Berikut adalah contoh penggambaran desain penelitian dalam bentuk diagramatik untuk penelitian tentang pengaruh penggunaan teknologi dalam pembelajaran matematika pada siswa sekolah dasar:

[Diagramatik]

Penjelasan:

- Variabel Bebas: Merupakan variabel yang dimanipulasi oleh peneliti, yaitu penggunaan teknologi dalam pembelajaran matematika oleh siswa.

- Variabel Tergantung: Merupakan variabel yang diukur atau diamati untuk melihat pengaruh variabel bebas terhadapnya, yaitu prestasi belajar matematika siswa.

- Variabel Kontrol: Merupakan variabel-variabel lain yang dapat memengaruhi variabel tergantung dan perlu dikontrol agar tidak mempengaruhi hasil penelitian secara signifikan.

Dalam penelitian ini, peneliti akan memilih dua kelompok siswa sekolah dasar. Satu kelompok akan menggunakan teknologi dalam pembelajaran matematika, sedangkan kelompok lainnya menggunakan metode pengajaran konvensional. Peneliti akan mengukur prestasi belajar matematika kedua kelompok tersebut setelah periode tertentu. Selain itu, peneliti juga akan mempertimbangkan variabel kontrol seperti latar belakang siswa, tingkat kemampuan awal, dan faktor-faktor lain yang dapat memengaruhi hasil penelitian.

Dengan desain penelitian ini, peneliti dapat mengidentifikasi apakah penggunaan teknologi dalam pembelajaran matematika memiliki pengaruh signifikan terhadap prestasi belajar siswa. Diagramatik ini membantu menggambarkan hubungan antara variabel-variabel dalam penelitian dan mengilustrasikan langkah-langkah yang akan diambil dalam penelitian.

29. Buatlah ilustrasi denah kamar sudah kamu tentukan pada AutoCAD. ada yang bisa mnggunakan aplikasi autocad ga di hp android. coba berikan contoh nya. atau tturial nya . makasih​


Jawaban:

Aplikasi AutoCAD tidak tersedia dalam bentuk resmi di perangkat Android. Namun, ada beberapa alternatif dan aplikasi serupa yang dapat Anda gunakan untuk membuat ilustrasi denah kamar atau desain lainnya di perangkat Android. Salah satu aplikasi yang populer adalah "AutoCAD - DWG Viewer & Editor."

Berikut adalah langkah-langkah umum untuk menggunakan AutoCAD-style aplikasi di perangkat Android:

1. **Instal Aplikasi AutoCAD Alternatif**: Unduh dan instal aplikasi AutoCAD alternatif seperti "AutoCAD - DWG Viewer & Editor" atau "DWG FastView" dari Google Play Store.

2. **Buat Proyek Baru**: Buka aplikasi, dan buat proyek baru. Biasanya, Anda dapat memilih opsi "New" atau "Buat Proyek Baru."

3. **Gambar Denah Kamar**:

- Gunakan alat gambar (seperti garis, persegi, atau lingkaran) untuk membuat denah kamar sesuai dengan skala yang diinginkan.

- Anda dapat menggunakan alat zoom untuk melihat atau memperbesar area tertentu.

4. **Tambahkan Detail**: Tambahkan detail seperti pintu, jendela, furnitur, dan lain-lain sesuai kebutuhan.

38. Sebutkan perintah dasar menggambar pada Autocad 2 dimensi


Jawaban:

Perintah Line: digunakan untuk membentuk objek garis pada layar kerja. Untuk menggunakannya anda dapat mengetikkan L atau Line (enter)

autocad-perintah-line

Perintah Circle: Digunakan untuk membuat objek lingkaran dengan beberapa mode mulai dari 2 point, center point dan tangent. Untuk menggunakannya ketik C atau Circle (enter)

autocad-shortcut-circle

Perintah ARC: Digunakan untuk membuat objek arc atau lingkaran tidak penuh. Untuk menggunakannya ketik ARC (enter)

autocad-shortcut-arc

Perintah Fillet: Digunakan untuk membentuk sudut yang tumpul ditentukan dengan jari-jari ataupun dimensi lingkaran. Untuk menggunakannya ketik Fillet (enter)

autocad-shortcut-fillet

Perintah Chamfer: Digunakan untuk membentuk sudut taper yang ditentukan dengan 2 ukuran berdasarkan sumbu x dan y. Ketik chamfer (enter) untuk menggunakannya

autocad-shortcut-chamfer

Perintah Trim: Digunakan untuk memotong objek yang saling bersinggungan antara satu dengan yang lainnya. Ketik tr atau trim (enter) untuk menggunakannya

autocad-shortcut-trim

Perintah offset: Digunakan untuk membuat duplikasi objek secara offset. Ketik offset (enter) untuk menggunakannya

autocad-shortcut-offset

Perintah mirror: Digunakan untuk melakukan duplikasi objek secara cepat dengan efek seperti cermin (sebaliknya). Ketika Mi (enter) untuk menggunakannya

autocad-shortcut-mirror

Perintah Copy: digunakan untuk menduplikasi objek ditentukan dengan titik pertama dan titik tujuan. Ketik Co (enter) untuk menggunakannya
