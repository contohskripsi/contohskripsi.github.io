16. jelaskan perbedaan antara WWW, HTML, dan HTTP


HTML (Hyper Text Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser Internet. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan di dunia penerbitan dan percetakan yang disebut dengan SGML (Standard Generalized Markup Language), HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

HTML berupa kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan. Sebuah file yang merupakan file HTML dapat dibuka dengan menggunakan browser web seperti Mozilla Firefox atau Microsoft Internet Explorer. HTML juga dapat dikenali oleh aplikasi pembuka email ataupun dari PDA dan program lain yang memiliki kemampuan browser.


HTTP Singkatan dari Hypertext Transfer Protocol, yang mana adalah suatu protokol yang digunakan oleh World Wide Web. HTTP mendefinisikan bagaimana suatu pesan bisa diformat dan dikirimkan dari server ke client.

HTTP juga mengatur aksi-aksi apa saja yang harus dilakukan oleh web server dan juga web browser sebagai respon atas perintah-perintah yang ada pada protokol HTTP ini.
Sebagai contoh, ketika Anda mengetikkan suatu alamat atau URL pada internet browser Anda, maka sebenarnya web browser akan mengirimkan perintah HTTP ke web server.

Web server kemudian akan menerima perintah ini dan melakukan aktivitas sesuai dengan perintah yang diminta oleh web browser (misalnya akses ke database, file, e-mail dan lain sebagainya). Hasil aktivitas tadi akan dikirimkan kembali ke web browser untuk ditampilkan kepada pengguna.

21. Buatkan contoh teks eksposisi yang berisi judul, tesis, agrumentasi, dan kesimpulan


Berikut ini contoh teks eksposisi:

                      "Menjaga Kebersihan Lingkungan Sekolah"

Tesis:

Salah satu faktor penting dalam menciptakan kenyamanan belajar ialah kebersihan lingkungan sekolah. Setiap sekolah pasti selalu mengajarkan tentang menjaga kebersihan lingkungan sekolah kepada anak didiknya. Salah satu cara mengajarkan anak didik mengenai kebersihan lingkungan sekolah ialah dengan memberitahu pentingnya membuang sampah pada tempatnya.

Rangkaian argumen:

Cara lainnya untuk menjaga kebersihan lingkungan sekolah ialah dengan menerapkan pembagian piket kelas. Petugas piket bertugas untuk membersihkan ruang kelas, seperti menyapu kelas, membuang sampah, menghapus papan tulis serta menyiapkan spidol atau kapur tulis. Setiap hari jumat juga siswa selalu diajarkan untuk melakukan kerja bakti membersihkan sekolah setelah pelajaran pertama selesai. Dari kegiatan tersebut banyak sekali manfaat yang diperoleh yaitu lingkungan menjadi bersih dan menumbuhkan solidaritas antar siswa.

Penegasan ulang:

Mengajarkan siswa mengenai menjaga kebersihan lingkungan sekolah sangatlah penting. Hal ini juga dapat membuat siswa merasa nyaman saat di sekolah dan belajar karena lingkungan yang bersih.

Pembahasan

Teks eksposisi merupakan teks yang menjelaskan suatu informasi atau pengetahuan kepada pembaca. Struktur teks tersebut ialah tesis, rangkaian argumen dan penegasan ulang.

36. Apakah perbedaan html Dan http?


Perbedaan HTML dan HTTP

»HTML (Hyper Text Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser Internet. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan di dunia penerbitan dan percetakan yang disebut dengan SGML (Standard Generalized Markup Language), HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).
HTML berupa kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan. Sebuah file yang merupakan file HTML dapat dibuka dengan menggunakan browser web seperti Mozilla Firefox atau Microsoft Internet Explorer. HTML juga dapat dikenali oleh aplikasi pembuka email ataupun dari PDA dan program lain yang memiliki kemampuan browser.


»HTTP Singkatan dari Hypertext Transfer Protocol, yang mana adalah suatu protokol yang digunakan oleh World Wide Web. HTTP mendefinisikan bagaimana suatu pesan bisa diformat dan dikirimkan dari server ke client.
HTTP juga mengatur aksi-aksi apa saja yang harus dilakukan oleh web server dan juga web browser sebagai respon atas perintah-perintah yang ada pada protokol HTTP ini. Sebagai contoh, ketika Anda mengetikkan suatu alamat atau URL pada internet browser Anda, maka sebenarnya web browser akan mengirimkan perintah HTTP ke web server.
Web server kemudian akan menerima perintah ini dan melakukan aktivitas sesuai dengan perintah yang diminta oleh web browser (misalnya akses ke database, file, e-mail dan lain sebagainya). Hasil aktivitas tadi akan dikirimkan kembali ke web browser untuk ditampilkan kepada pengguna.HTTP itu adalah sebuah protokol jaringan untuk pengaksesan konten2 dalam internet, dan
