27. mohon dijawab dgn baik dan bnar​


Penjelasan dengan langkah-langkah:

deret bilangan..

rumus un ---> Un = ½n² + ⅔

= u¹²

= ½n² + ⅔

= ½ (12)² + ⅔

= ½ (144) + ⅔

= 72 + ⅔

= (216 + 2)/3

= 218/3 (B)

Nilai suku ke-12 barisan tersebut ialah 218/3 [Opsi B]

PEMBAHASAN

Un = ½n² + ⅔

substitusikan U12 kerumus, diperoleh:

Un = ½n² + ⅔

U12 = ½ (12)² + ⅔

= ½ (144) + ⅔

= 72 + ⅔

= {(72 . 3) + 2}/3

= {216 + 2}/3

= 218/3

=============================
