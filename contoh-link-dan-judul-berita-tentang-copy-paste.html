Namun, aturan tersebut dikecualikan untuk kategori masyarakat yang tidak bisa melakukan vaksinasi.

Saat mendatangi apotek, masyarakat perlu membawa sertifikat vaksin COVID-19.

Tidak perlu khawatir, tempat ini buka selama 24 jam dalam sehari.

Buka software Adobe Acrobat, klik tombol Create yang ada di toolbar utama. Jika Anda tidak menemukan tombol ‘Create PDF’, buka View yang ada di toolbar, lalu pilih tombol Task dan lanjutkan dengan memilih Show All Task Buttons.

Proses penggabungan dokumen menjadi satu file PDF telah selesai, kini saatnya Anda menyimpan file PDF di ‘Save As..’ dengan nama yang Anda inginkan.

Copy Data Workbook.

Cara lain untuk melakukan copy data dari workbook ke dokumen lainnya adalah dengan menggunakan opsi "Paste Values".

Dengan menggunakan opsi ini, data yang disalin akan dipasang ke dokumen baru dengan nilai yang sama seperti di workbook aslinya, namun tanpa format atau rumus yang mungkin ada di workbook.

Anda dapat menggunakan opsi "Paste Values" dengan cara menekan tombol "Ctrl + Alt + V" pada keyboard, atau melalui menu "Paste Special" di tab "Home" pada Ribbon, dan memilih opsi "Values".

Kalau cut + paste, kamu hanya memindahkan lokasi file aslinya, bukan menduplikasi
