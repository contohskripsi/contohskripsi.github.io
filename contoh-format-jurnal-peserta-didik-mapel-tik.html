22. contoh batasan masalah TIK dalam bidang pendidikan?


Pengertian Teknologi Informasi dan Komunikasi
Teknologi informasi dan komunikasi mempunyai pengertian dari dua aspek, yaitu Teknologi Informasi dan Teknologi Komunikasi. Teknologi Informasi mempunyai pengertian luas yang meliputi segala hal yang berkaitan dengan proses, penggunaan sebagai alat bantu, manipulasidan pengolahan informasi. Teknologi Komunikasi mempunyai pengertian segala hal yang berkaitan dengan penggunaan alat bantu untuk memproses dan mentransfer data dari perangkat yang satu ke yang lain. Karena itu Teknologi Informasi dan Komunikasi adalah suatu padanan yang tidak terpisahkan yang mengandung pengertian luas tentang segala aspek yang berkaitan dengan pemprosesan, manipulasi, pengolahan, dan transfer/ pemindahan informasi antara media menggunakan teknologi tertentu.

Apa Itu CMS?

CMS (Content Manajemen System) adalah sebuah perangkat lunak atau sistem yang mengatur konten pada situs web, bisa situs web yang berupa company profile, e-commerce, blog, forum, dan lainnya.

5 contoh CMS beserta penjelasannya?1. WordPress

WordPress merupakan salah satu CMS website terbaik yang sering orang-orang gunakan. WordPress ini bersifat open source, sehingga semua orang bisa berkontribusi dalam membuat tools atau plugin untuk CMS yang satu ini.

2. Drupal

Selanjutnya ada juga salah satu dari berbagai macam CMS untuk membuat website yang sedang populer, yaitu Drupal.

3. Joomla

Joomla ini sama sama menggunakan lisensi GPL dan juga masih bersifat open source seperti yang dimiliki Drupal tadi.

4. Prestashop

Prestashop merupakan salah satu CMS untuk membuat toko online.

5. OpenCart

Dari berbagai jenis CMS website untuk kebutuhan e-commerce, OpenCart menjadi salah satu pilihan yang tepat. CMS OpenCart ini banyak digunakan bagi kalangan personal maupun bagi UKM yang ingin membangun sebuah toko online.

40. sebutkan dan jelaskan format format untuk halaman pljrn tik niiih


JPG/JPEG

JPG/JPEG singkatan dari Joint Photographic Expert Group. Biasanya format file ini mampu mengkompres objek dengan tingkat kualitas sesuai dengan pilihan yang telah disediakan. Format file sering dimanfaatkan untuk menyimpan gambar yang akan digunakan untuk keperluan halaman multimedia, web dan meida elektronik lainnya. Perlu teman-teman ketahui bahwa format ini bisa menyimpan gambar dengan mode warna CMYK, RGB dan Grayscale.

PNG

PNG singkatan dari Portable Network Graphic. Format file ini berfungsi untuk menampilkan objek dalam halaman web. Kelebihan dari format file ini adalah menghasilkan latar belakang atau biasa disebut background yang transparan dengan pinggiran yang halus.

PSD

PSD adalah singkatan dari Photoshop Document. Format file ini merupakan format asli dokumen Adobe Photoshop. Format ini mampu menyimpan berbagai macam layer, sehingga suatu saat dapat dibuka dan diedit kembali. Format ini juga mampu menyimpan gambar dalam beberapa mode warna yang disediakan Photoshop. Anda dapat menyimpan dengan format file ini jika ingin mengeditnya kembali dan tidak membuatnya dari awal.

TIF

TIF adalah singkatan dari Tagged Image Format File. Format file ini mampu menyimpan gambar dengan kualitas hingga 32 bit. Format file ini merupakan salah satu format yang dipilih dan sangat disukai oleh para pengguna komputer grafis terutama yang bergelut dibidang percetakan. Karena hampir semua program yang mampu membaca format file bitmap juga mampu membaca format file TIF.

PDF

PDF adalah singkatan dari Portable Document Format. Format file ini mampu menyimpan gambar dengan mode warna RGB, CMYK, Indexed Color, Lab Color, Grayscale dan Bitmap. Format file ini tidak mampu menyimpan alpha channel. Format file ini sering menggunakan kompresi JPG dan ZIP, kecuali untuk mode warna Bitmap yaitu menggunakan CCIT.
